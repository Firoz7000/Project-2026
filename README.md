# Project-2026
# 🛒 E-Commerce Website (Frontend Project)

## 📌 Project Overview

This project is a simple e-commerce website developed using frontend technologies.
It demonstrates the basic functionality of an online shopping platform, including product display, search, filtering, and cart management.

---

## 🎯 Objectives

* To understand the structure of a web application
* To design a responsive and user-friendly interface
* To implement basic interactivity using JavaScript

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ✨ Features

* 🔍 Product Search functionality
* 🎯 Category-based filtering (Electronics, Fashion, Grocery)
* 🛒 Add to Cart system
* 🧾 Dynamic Cart with total calculation
* 💬 Feedback form (UI-based)
* 📱 Responsive design for different screen sizes

---

## 📂 Project Structure

```
Project-2026/
│── index.html

│── images/

```

---

## 🚀 How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Run the `index.html` file in any web browser
